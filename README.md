# Modern Responsive Agency Website

A sleek, fully responsive business/agency website built using React.js, JavaScript (ES6+), and Tailwind CSS.

## Features
- Modern, professional UI suitable for real-world business use
- Fully responsive — optimized for mobile, tablet, and desktop
- Built with reusable React components for scalability
- Utility-first styling with Tailwind CSS

## Tech Stack
- React.js
- JavaScript (ES6+)
- Tailwind CSS

## Getting Started
npm install
npm run dev
